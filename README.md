Описание программы

Графическое приложение для генерации криптографически стойких случайных паролей. Позволяет гибко настраивать сложность пароля и вести историю последних сгенерированных значений.
Технологии и зависимости

    Язык: Python 3.x.
    GUI: tkinter (стандартная библиотека).
    Генерация: random (стандартная библиотека).
    Хранение данных: json (стандартная библиотека).
    Опционально: ttkthemes для красивого оформления.
