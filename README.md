# Weather Diary

**Автор:** Иван Иванов

**Описание:** Простое приложение для ведения дневника погоды с возможностью добавления, фильтрации и сохранения записей.

**Как запустить:**
1. Установите Python 3.x.
2. Установите зависимости: `pip install tk`
3. Запустите: `python weather_diary.py`

**Примеры использования:**
- Добавьте запись о погоде за сегодня.
- Сохраните данные в файл weather.json.
- Загрузите данные из файла.
